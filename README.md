# 校园导师模拟器

一个基于React + TypeScript的校园导师职业生涯模拟游戏，体验真实的学术生涯发展过程。

## 🎮 游戏特色

### 核心玩法
- **年度决策制**：每年只能选择一个事件、购买一件装备、申请一个项目
- **学生管理系统**：学生每3年毕业，根据忠诚度招募新学生
- **科研项目申请**：不同项目有不同的属性要求和奖励
- **五年评估机制**："非升即走"制度，未达标则游戏结束

### 属性系统
- **学术分数**：代表学术水平和研究成果
- **科研经费**：用于购买装备和维持研究
- **学术声誉**：影响项目申请和学生招募
- **学生忠诚度**：影响学生工作效率和新学生质量

### 游戏机制
- **装备效果**：提升学生工作效率和忠诚度（非直接属性加成）
- **项目奖励**：完成项目获得属性奖励和学生工作效率提升
- **学生毕业**：毕业生会根据潜力提供学术分数和声誉奖励
- **职称晋升**：助教→讲师→副教授→教授，每级有不同的评估标准

## 🚀 技术栈

- **前端框架**：React 18 + TypeScript
- **状态管理**：Zustand（支持本地存储）
- **样式方案**：Tailwind CSS
- **构建工具**：Vite
- **路由管理**：React Router DOM

## 📦 项目结构

```
src/
├── components/          # 可复用组件

│   ├── projects.ts     # 科研项目数据

│   └── evaluation.ts   # 评估标准数据
├── pages/              # 页面组件
│   ├── MainGame.tsx    # 主游戏界面
│   ├── StudentManagement.tsx  # 学生管理
│   ├── ProjectApplication.tsx # 项目申请
│   ├── EquipmentStore.tsx     # 装备商店
│   ├── EvaluationSystem.tsx   # 评估系统
│   └── GameOver.tsx    # 游戏结束界面
├── store/              # 状态管理
│   └── gameStore.ts    # Zustand store
├── types/              # TypeScript类型定义
│   └── game.ts        # 游戏核心类型
└── App.tsx            # 根组件
```

## 🎯 游戏玩法

### 1. 主界面
- 查看当前角色状态和属性
- 了解年度任务完成情况
- 查看进行中的项目进度
- 进入下一年（需完成所有年度任务）

### 2. 学生管理
- 查看所有学生的详细信息
- 选择年度学术事件
- 监控学生毕业进度

### 3. 项目申请
- 浏览可申请的科研项目
- 查看项目要求和奖励
- 申请符合条件的项目

### 4. 装备商店
- 购买提升学生效率的装备
- 管理实验室资源和设施
- 合理分配科研经费

### 5. 五年评估
- 查看当前评估进度
- 了解各项属性达标情况
- 获得短期和长期发展建议

## 🎮 游戏策略建议

### 初期发展（1-3年）
1. **优先提升学生忠诚度**：通过装备和事件提升
2. **申请简单项目**：积累初期资源和经验
3. **合理购买装备**：优先提升学生工作效率

### 中期发展（4-8年）
1. **平衡属性发展**：确保各项属性均衡发展
2. **申请中等难度项目**：获得更好的奖励
3. **关注评估要求**：提前准备五年评估

### 后期发展（9年以上）
1. **挑战高难度项目**：获得丰厚奖励
2. **维持高水平属性**：确保通过评估
3. **培养优秀学生团队**：为长期发展做准备

## 🔧 开发说明

### 状态管理
使用Zustand进行状态管理，支持本地存储持久化：
- 游戏状态自动保存到localStorage
- 支持游戏重新开始和状态恢复
- 响应式状态更新

### 数据配置
所有游戏数据都集中在`src/data/`目录：
- 易于修改和扩展
- 类型安全的配置
- 支持动态生成内容

### 组件设计
- 功能单一的组件设计
- 响应式布局支持
- 美观的渐变色彩和动画效果

## 📱 响应式设计

- 优化的移动端体验
